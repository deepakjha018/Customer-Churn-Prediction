# Customer Churn Prediction

## Overview

This project predicts whether a telecom customer is likely to churn based on demographic information, account details, and subscribed services using Machine Learning.

## Project Structure

- Data Collection
- Exploratory Data Analysis
- Data Preprocessing
- Feature Engineering
- Model Training
- Model Evaluation
- Streamlit Deployment

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit
