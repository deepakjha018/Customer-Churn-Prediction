# Customer Churn Prediction Evaluation Report

## Dataset

IBM Telco Customer Churn Dataset

Total Customers: 7043

Training Samples: 5625

Testing Samples: 1407

---

## Models Compared

- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- Hist Gradient Boosting

---

## Best Model

Gradient Boosting

Accuracy : 77%

ROC-AUC : 0.84

Precision : 0.56

Recall : 0.69

F1 Score : 0.62

---

## Conclusion

Gradient Boosting achieved the highest balance between recall and overall performance and was selected for deployment.
